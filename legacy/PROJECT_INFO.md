# Project Info

QiCompute is a privacy-first experimental distributed inference marketplace prototype with useful-work verification and settlement accounting.

Version: 0.1.0

License: MIT License. See `LICENSE`.

## Goals

- Make private distributed inference measurable and testable locally.
- Verify useful compute with receipts, challenges, committees, and audit trails.
- Model settlement accounting before integrating any real payment or blockchain layer.
- Keep home GPU operator workflows simple enough for LAN experimentation.
- Preserve a path toward decentralization without overclaiming current capabilities.

## Non-Goals

- Public internet networking.
- Production cryptography.
- Blockchain integration.
- Wallets or token transfers.
- Cloud services.
- Production consensus.

## Design Principles

- Privacy first.
- Deterministic accounting.
- Useful-work verification.
- Local-first development.
- Decentralization readiness.
- Simulation before irreversible protocol commitments.
